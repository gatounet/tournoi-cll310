    Informations du tournoi

    La connexion admin est sécurisée par Firebase Authentication. Pour ajouter ou changer un compte admin, rends-toi dans la console Firebase → Authentication → Users.

    Importer une sauvegarde

    Réimporte un fichier de sauvegarde JSON précédemment téléchargé pour restaurer équipes, matchs, groupes et terrains.

    L'import ajoute les données du fichier à celles déjà présentes. Pour repartir d'une base vide, réinitialise d'abord le tournoi (ci-dessous).

    Zone dangereuse

    Sauvegarde toutes les données actuelles (équipes, matchs, groupes, terrains) dans un fichier que tu peux conserver, puis réinitialise le tournoi pour repartir de zéro.

    🪄Générer les matchs automatiquement

    Comment ça marche ?

    Tous les matchs possibles entre les équipes de chaque groupe seront générés (chaque équipe rencontre toutes les autres). Tu peux choisir le type de phase et le terrain.

    🏆 Victoire = 2 pts 🤝 Nul = 1 pt ❌ Défaite = 0 pt
    L'ordre des matchs est automatiquement réorganisé pour qu'une équipe ne joue jamais deux matchs consécutifs.

    ⚠️Réinitialiser le tournoi

    Cette action est irréversible

    Toutes les équipes, matchs, groupes et terrains seront définitivement supprimés de la base de données. Les paramètres du tournoi (nom, sous-titre) seront conservés.
